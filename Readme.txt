# Overview
- A responsive web page built with **Bootstrap 5** as part of Module 5 assignment. It includes a registration form, user table, image section, buttons, and a responsive navbar.

# Features
- Registration Form** with validation and grid layout
- User Table** with striped, hoverable rows inside a responsive wrapper
- Image Section** with a full-width banner and circular image
- Utility Buttons**: one always visible, one hidden on small screens
- Navbar** with Home, About, Contact links and hamburger toggle.
